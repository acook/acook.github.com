Directory Structure

I cloned the site repo into an ignored subdirectory (called public):

$ git clone . public

Then I changed the branch and cleaned up the subdirectory:

$ git co -b develop
$ cd public
$ shopt -s extglob
$ git rm -rf .
$ cd ..
$ hugo
$ cd public
$ git add .
$ git commit
