INSTALLING RUBY 1.9.2, RAILS 3, AND MYSQL ON WINDOWS

Ruby

  1. Download the current Ruby Installer for Windows:
  2. http://rubyforge.org/frs/download.php/74298/rubyinstaller-1.9.2-p180.exe
  3. or check for latest at http://rubyinstaller.org/downloads/
  4. Run the installer
  5. Select
  6. Download the DevKit
  7. sfsf
  8. adfkljbdf

MySQL

  1. Download the MySQL Installer for Windows:
  2. current: http://dev.mysql.com/downloads/mirror.php?id=402461
  3. all: http://dev.mysql.com/downloads/mysql/
  4. run installer
  5. copy “C:\Program Files\MySQL\MySQL Server 5.1\libmysql.dll” C:\ruby19\bin
