Get the delete key working in IRB on OSX with two lines of code

23/Aug 2011 1 min. read

Its been driving me nuts. The delete key doesn’t work in IRB (or Pry) on OSX! ed: I mean forward-delete not backspace: function+delete on my work MBP and the normal delete key above the arrows on my shiny Das Keyboard. Anyway, today I took a couple of minutes and tried this: brew install readline rvm install 1.9.2 --with-readline-dir=/usr/local/Cellar/readline/6.2.1 It works now. That’s seriously all I had to do. No patching required!
